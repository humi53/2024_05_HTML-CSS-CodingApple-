# Table 레이아웃 숙제 : Cart 페이지 만들기

### table 태그에 border-radius가 안먹을 때.

```css
table {
  border-collapse: collapse;
  border-spacing: 0;
}

(왼쪽위에있는 td) {
  border-top-left-radius: 5px;
}
```

```css
table {
border-collapse : collapse;
border-radius : 7px;
border-style : hidden;
box-shadow : 0 0 0 1px #666;
}
```
