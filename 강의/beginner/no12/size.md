# size

### 숙제

```
.explain-box {
  background: #eee;
  padding: 20px;
  position: relative;
  left: 0;
  right: 0;
  margin: auto;
  top: 160px;
  width: 600px;
}
```

### 배운것

#### 앞뒤

- `z-index: 1;`
- 곂침 앞뒤로 보내기

#### 반응형

```
    width: 80%;
    max-width: 600px;
```

- max-height, min-width

#### 사이즈를 통틀어 결정.

- padding, margin : 이런것들을 함께 사이즈 계산
- `box-sizing: border-box;'

### aa

- normalize.css : 브라우저간 호환성 잡는 파일 제공.
